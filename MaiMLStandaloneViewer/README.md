# MaiMLStandaloneViewer

**MaiMLファイルをブラウザで可視化できる単一HTML Viewer**  
サーバー不要・インストール不要で、1つのHTMLファイルだけで動作します。

MaiMLファイル（`.maiml` or `.maiml.zip`）をブラウザにドラッグ＆ドロップするだけで、  
構造や内容を可視化できます。

---

## 🚀 特徴

- **MaiMLファイル**（`.maiml`）をブラウザ上で可視化  
- **単一HTMLファイル構成** 
- **ペトリネット図の可視化**
- **テンプレート、インスタンス部分のデータを確認**可能
- **外部データの可視化**に対応
- OSに依存せず、macOS / Windows / Linux すべてで動作
- Gemini APIを用いた**要約機能**

---

## 🧩 使い方

1. `HTML-MaiMLViewer.html` をダウンロード  
2. ブラウザ（Chrome / Safari / Edge など）で開く  
3. `.maiml` ファイルをドラッグ＆ドロップ  
4. 外部ファイルをアップロードして可視化
5. content要素（２次元）をグラフ表示
6. material/condition/resultのコンテンツの要約を表示

> 💡 WEBサーバーは不要です。

---

## ⚠️ 注意

1. ファイルサイズが大きい場合、ブラウザがクラッシュすることがあります
