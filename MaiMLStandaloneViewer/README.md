# MaiMLStandaloneViewer

MaiML (Measurement Analysis Instrument Markup Language) ファイルを可視化・分析するためのスタンドアロン Webアプリケーション。

## 🌟 主な機能

### 📊 グラフ可視化
- **Petri Net ベースのワークフロー可視化**：実験プロセスの状態遷移をノードグラフで表示
  - **Place（円）**：試料やデータの状態を表現
  - **Transition（長方形）**：処理やイベントを表現
- **参照関係の可視化**：（ツールボックスで表示On/Off可）
  - **Template Ref（黄色の破線）**：テンプレート同士の参照関係
  - **Instance Ref（緑色の破線）**：インスタンス同士の参照関係
  - **Chain Link（橙色の破線）**：チェーン接続関係の可視化
  - **Same Uuid（青色の破線）** : 同一UUIDの可視化（テンプレート・インスタンス）

### 📄 メタデータ管理と表示
- **ドキュメント情報パネル**：作成者、UUID、説明、ベンダー情報の一元表示
- **詳細パネル**：選択ノードの関連テンプレート・インスタンス情報を表示
- **テンプレート・インスタンスの階層表示**：ノード選択で該当データを自動抽出

### 📁 ファイル対応
- **.maiml ファイル対応**：MaiML 標準フォーマットを完全サポート
- **.zip アーカイブ対応**：ZIP内のMaiMLファイルと関連画像/データを自動リンク
- **埋め込みファイルプレビュー**：
  - 画像（JPG, PNG, TIFF等）の直接表示
  - CSV/テキストデータのテーブル表示
  - TIFF マルチページ対応

### 🎨 UI/UX 機能
- **ライト/ダークテーマ対応**：CSS変数によるシームレスな切り替え
- **レスポンシブデ���イン**：水平・垂直リサイザーでパネルサイズを調整
- **グラフ操作**：
  - マウスホイールによるズーム（0.25 ～ 6倍）
  - ドラッグによるパン移動
  - ノード手動配置（Drag Edit チェックボックス）

### 🎯 レイアウト機能
- **Force-directed Layout**：力学モデルによる自動配置
- **Dagre Hierarchical Layout**：階層的な自動整理
- **グリッド表示/スナップ**：整列・配置補助機能

### 🤖 AI 分析機能
- **Google Gemini API統合**：詳細データの AI 要約
- **複数プロンプトテンプレート**：
  - Quick Summary
  - Standard Summary
  - Detailed Report
  - Custom Prompt（ユーザー定義）
- **マークダウン形式で結果表示**：保存・コピー機能完備

### 📊 データプロット機能
- **3種類のグラフ形式**：
  - **Line（折れ線）**：時系列トレンド表示
  - **Bar（棒グラフ）**：ヒストグラム表示
  - **Scatter（散布）**：相関分析表示
- **インタラクティブ操作**：ズーム、パン、リセット機能
- Chart.js + チャートズームプラグイン統合

### 📈 メタデータエクスポート
複数形式でのデータ出力対応：
- **Excel (.xlsx)**：シート構成による一覧表示
- **JSON (.json)**：階層構造を維持した生データ
- **CSV (.csv)**：汎用ツール連携用

### 🔐 XML 暗号化データ対応（JIS K0200 準拠）
- **AES-256-GCM + PBKDF2-SHA256 復号対応**：W3C XML Encryption 1.1 準拠
- **暗号化バナー通知**：検出時に自動表示
- **一括復号機能**：「Decrypt All」ボタンで全要素を一度に復号
- **個別復号オプション**：ノードごとの段階的復号
- **パスワードキャッシング**：セッション中の自動再利用
- ⚠️ **制限**：パスワード方式のみ対応（RSA/鍵ラップ方式は未対応）
